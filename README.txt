# MV_MEV-thesis
 This Repo contains code and data used for the Master thesis as part of the MScBA Business Analytics and Management at the Rotterdam School of Management, Erasmus University. Preliminary title: 'Analyzing the impact of Sandwich attacks as a form of Miner Extractable Value on network efficiency on the Ethereum Blockchain'. The author of this thesis is Marc Vendramet. This repo contains:

01: JupyterNotebooks with python code for data collection, preparation, and attack identification
02: R Markdown file with R code for data analysis, visualization, and regressions
03: .txt file containing SQL code for data collection on GoogleBigQuery
